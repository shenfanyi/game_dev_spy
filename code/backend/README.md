## Getting Started

- Create Virtual Environment

  [Tips](http://flask.pocoo.org/docs/1.0/installation/#virtual-environments)

  ```shell
  $ python -m venv venv
  $ . venv/bin/activate
  ```

- Install Flask

  [Tips](http://flask.pocoo.org/docs/1.0/installation/#install-flask)

  ```shell
  $ pip install Flask
  ```

- Run ``start.sh``
